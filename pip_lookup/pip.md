# PIP - Python Package manager
> PIP (Pip Installs Packages or Pip Installs Python) is the official package manager for Python.
It allows developers to install, upgrade, and manage third-party libraries and dependencies with ease.
PIP automates the process of handling Python modules, so you don’t need to manually download or configure them.

## Commands:
- `pip help`: Detail description about commands.

- `pip help <specific-command>`: Shows a detail description about that specific command.

- `pip search <package-name>`: As the name says for searching a package it gives the package description.

- `pip install <package-name>`: For installing the package.

- `pip list`: For taking a look at the installed packages.

- `pip uninstall <package-name>`: For uninstalling the package that we don't want.

- `pip list --outdated` / `pip list -o`: For listing the packages with their current version that was installed and the latest version out there.

- `pip install -U <package-name>`: For updating a package.

- `pip freeze`: If you are working on a project and you're project has a list of packages and you want to give those packages to your teammate then this is how you can give them.

- `pip freeze > requirements.txt`: You can also do this.

- `pip install -r <file.extension>`: If we have a list of packages and we have to install them all through a file.

> These are the commands that you usually use.
